# @fe-hl/shared

## javascript 常用工具库，支持 typescript 日期、金额、本地存储、常用正则、其他工具等等

# 使用说明

> javascript 常用工具库，支持 typescript

## npm 安装

```sh
npm i @fe-hl/shared -s
```

## npm 打包

```sh
npm run build
```

### 导入方式一

```js
import * as shared from '@fe-hl/shared';
```

### 导入方式二

```js
import {
	dateAddDay,
	dateAddMonth,
	dateAddYears,
	dateDiffDay,
	dateDiffHour,
	dateDiffMonth,
	dateDiffYears,
	dateFormat,
	dateSubtractDay,
	dateSubtractMonth,
	dateSubtractYears,
	dateToTime,
	...
} from '@fe-hl/shared';
```
